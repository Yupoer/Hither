# 主題系統數據流 🆕

## DarkBlue 主題管理

```
ThemeManager → ThemeConfiguration → SwiftUI 組件
```

**流程：**
1. ThemeManager 偵測系統主題變更
2. ThemeConfiguration 提供 OKLCH 色彩值
3. SwiftUI 組件自動更新顏色
4. DarkBlue 組件應用一致樣式

**主題數據：**
```swift
struct ThemeColors {
    let background: Color
    let foreground: Color
    let primary: Color
    let secondary: Color
    // ... 完整的 CSS 設計令牌
}
```
